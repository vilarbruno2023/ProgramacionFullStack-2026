1) ¿Cómo adaptaré el Ejercicio 2.4 para mi futura página de Contacto?

Resumen: Usare la estructura básica del Ejercicio 2.4 y la convertiré en el formulario de contacto principal de mi portfolio.

  • Conservaré los campos esenciales nombre, correo, asunto, mensaje
  • Mejoraré la accesibilidad
  • Diseñare con CSS y pondre un diseño responsive

2) ¿Cómo reutilizaré el Ejercicio 2.8 como mi página "Sobre mí" definitiva?

Resumen: Utilizaré el contenido y estructura del Ejercicio 2.8 como base, detllandolo para mostrar experiencia, habilidades y cosas utiles

  • Transformaré el texto en secciones: resumen profesional, habilidades técnicas, proyectos destacados y educación
  • Añadiré un diseño interactivo: iconos de las habilidades y enlaces a repositorios
  • Haré que sea modular: cada proyecto será una tarjeta reutilizable (clase .proyecto) para mantener coherencia en todo el sitio
  • Incluiré una foto profesional optimizada y un CV en PDF descargable

3) ¿Qué estilos base del Ejercicio 2.6 mantendré para TODO mi portfolio?

Resumen: Mantendré una tipografía limpia, una paleta limitada y variables CSS para consistencia

  • Sistema de colores: variables CSS para color primario, fondo, texto y acentos (generalmente tonos pasteles)
  • Tipografía: Arial legible
  • Contenedor centrado consistente y border-radius uniforme
  • Componentes reutilizables: botones, tarjetas
  • Accesibilidad visual: contraste suficiente

Reflexión retrospectiva — ¿Cómo lo aprendido me servirá en el mercado laboral uruguayo?

La creacion de paginas web completas son directamente aplicables en puestos de desarrollo full-stack. De ejemplo: en un local que quiere tener su sitio online, podré implementar una página de contacto confiable, asi como un carrito de productos. Los clientes podran tener una cuenta para comprar y guardar sus productos. La pagina sera hecha con estilos reutilizables para mantener coherencia entre la web y reduciendo tiempos de creacion y costos de mantenimiento.
